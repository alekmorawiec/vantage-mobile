# Vantage Authentication v1

Adds Supabase email/password sign-up and login, session persistence, patient/provider routing, and logout.

Install dependencies:

```bash
npx expo install @supabase/supabase-js @react-native-async-storage/async-storage react-native-url-polyfill
```

Then unzip this package over the project and run:

```bash
npx expo start --clear
```
