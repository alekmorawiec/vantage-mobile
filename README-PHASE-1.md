# Vantage Phase 1

This package replaces the starter Expo screen with:

- Shared design system
- Reusable button, input, logo, and metric-card components
- Patient/provider role switch
- Demo login
- Patient dashboard
- Provider dashboard
- No new dependencies

## Install

From the root of the local Expo project:

```bash
cd ~/Development/vantage
unzip -o ~/Downloads/vantage-phase-1.zip
npx expo start
```

If the browser downloaded the file somewhere other than `~/Downloads`, drag the ZIP into the Downloads folder first.

## Demo credentials

The login form is intentionally not validated yet. Select either role and tap **Log in**.

Supabase authentication is the next phase.
